- 时间复杂度  （数量级）

  - 一条语句的重复执行次数称作**语句频度**(FrequencyCount)
  - **基本语句：执行次数最多；对算法运行时间贡献最大；嵌套最深的语句。**

  分析方法：

  ​	1.找出语句频度最大的那条语句作为基本语句；

  ​	2.计算基本语句的频度，得到问题规模n的某一个函数；

  ​	3.取其数量级用O表示
